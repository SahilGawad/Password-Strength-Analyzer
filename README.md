**Password Strength Analyzer** 

A responsive and user-friendly Password Strength Analyzer that helps users create secure passwords by checking strength in real-time and suggesting stronger alternatives.

**Features**

•	Live password strength detection

•	Weak, Medium, and Strong indicators

•	Dynamic strength progress bar

•	Password improvement suggestions

•	Strong password generator

•	Generate another password option

•	Show/Hide password feature

•	Clean and modern UI

•	Mobile responsive design


**Built With**
 
•	HTML

•	CSS

•	JavaScript



**Password Validation Checks**

The analyzer verifies:

•	Password length

•	Uppercase letters

•	Lowercase letters

•	Numbers

•	Special characters



**How to Use**

•	Enter a password

•	View password strength instantly

•	Read suggestions for improvement

•	Generate a secure password if needed

•	Toggle password visibility anytime
